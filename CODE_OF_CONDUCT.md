# Code of Conduct

## Our Pledge

OpenAmber is a project focused on child safety. We are committed to providing a welcoming, respectful, and harassment-free environment for everyone.

## Our Standards

### Expected Behavior

- Use welcoming and inclusive language
- Be respectful of differing viewpoints and experiences
- Gracefully accept constructive criticism
- Focus on what is best for the community and the project's mission
- Show empathy towards other community members
- Maintain appropriate gravity given the project's child safety focus

### Unacceptable Behavior

- Harassment, discrimination, or intimidation in any form
- Trolling, insulting/derogatory comments, and personal attacks
- Public or private harassment
- Publishing others' private information without permission
- Sharing sensitive information about real missing children cases
- Using project resources for purposes other than child safety
- Other conduct which could reasonably be considered inappropriate

## Scope

This Code of Conduct applies to all project spaces, including: 

- GitHub repositories (issues, PRs, discussions)
- Project communication channels
- Project events and meetups
- Representation of the project in public spaces
